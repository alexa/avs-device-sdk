Detailed Description

Class responsible for managing endpoints handled by this client. This includes:

  • Registering the endpoint and its capabilities with AVS (via CapabilitiesDelegateInterface).
  • Registering endpoint's directive handlers with DirectiveSequencerInterface.
  • Ensuring endpointId uniqueness across endpoints controlled by this device.
