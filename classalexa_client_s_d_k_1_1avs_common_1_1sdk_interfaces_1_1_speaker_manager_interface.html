pure virtual

Adds a speaker to be tracked by SpeakerManagerInterface. This method is not guaranteed to be thread safe and should be called during the initialization step only. All speakers added this way must be destroyed during the shutdown process.

Remarks
Note that after this method SpeakerManagerInterface instance will hold a reference to the SpeakerInterface added.
