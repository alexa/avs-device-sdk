Detailed Description

Define a class that can be used to run a function when the object goes out of scope.

This simulates try-finally statements. The following structure:

try {
<try_block>
} finally {
<finally_block>
}

can be replaced by:

FinallyGuard guard { []{ <finally_block> }};
<try_block>
