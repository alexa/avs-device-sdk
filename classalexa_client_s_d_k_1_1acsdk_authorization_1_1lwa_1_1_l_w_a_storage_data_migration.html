Detailed Description

Class to migrate storage database format to support SQLiteMiscStorage.

This class provides method to migrate data from existing instances of LWAAuthorizationStorage from a format before 1.26 release into a new one.

Migration allows end-users to continue using Alexa device without need to reauthorize it.
