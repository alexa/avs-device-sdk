Detailed Description

template<typename T>
class alexaClientSDK::avsCommon::utils::sds::SharedDataStream< T >::Writer

This is a nested class in SharedDataStream which provides an interface for writing (producing) data to the stream.

Note
This class is primarily intended to be used from a single thread. The Writer as a whole is thread-safe in the sense that Writer and Readers can all live in different threads, but individual member functions of a Reader instance should not be called from multiple threads except where specifically noted in function documentation below.
